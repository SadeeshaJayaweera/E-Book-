Online Mapping and Route Plan

Google maps

This is an online service which provides location services to the users. It provides maps of most of the places in this world with both aerial and satellite view. Most of the users installing this app to their smartphones because itâ€™s a free service. However this service provides detailed route to the user once he/she chose a destination to go.

Features like traffic information, time taken to the route,etc.. Are available with this service. Therefore anyone in this world can find any place they want by using this service rather than using huge map books.

However in order to use this service users must turned on the Global Positioning system (Gps) on their regular devices.

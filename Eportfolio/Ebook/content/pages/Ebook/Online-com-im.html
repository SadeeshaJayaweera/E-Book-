Communication

Instant Messaging

Instant messaging is a service that enables individuals to send quick text messages around the world, talk for free on the internet and transfer files. when a message is sent to you, it appears up on your screen. So you can reply if you wish. When you are online you can see the chat list which shows active and inactive friends. This can be done using a browser.

Instant messaging is mostly used in email services like Google. This can be done using the browser or you can use the application like GOOGLE PLUS. Now a days most of the people stepping into mobile applications like WHATSAPP AND VIBER in order to make group and individual messaging easy. which can be installed in smartphones. These applications likewise have the features of instant messaging.

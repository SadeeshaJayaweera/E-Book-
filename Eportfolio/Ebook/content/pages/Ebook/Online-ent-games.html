Multi – User Games

Multi-user games are games that allow a user to play the same game with multiple players through the internet.

There are many games available on the internet which can be played directly. Friv.com is a good example as it provides a huge number of games in many categories.so users can select and play games that they want. However, if the users want to play with others so they can play by joining a group.

Games like call of duty is a good example of multi-user game. This Game comes with separate modes called the single player and multiplayer. In order to use this service player must installed the game into their computer and need an internet connection too.

These days online games became very popular among smartphone users. Teenagers addicted to playing games like “PUBG” and “Clash_of_Clans". However, this is not good because they are neglecting their studies and playing games constantly.
